| 最小间距 | 0.1mm (4mil) |

| 表面处理 | ENIG (沉金) |
| 板材 | FR4 (Tg≥170°C) |
| 介电常数 | 4.3 ± 0.2 |

### 7.2 焊接工艺

**DW3220 QFN 焊接**:
- 回流焊温度曲线: 峰值 245°C
- 焊盘设计: 1:1 或略小于引脚
- 中心散热焊盘: 多过孔连接 GND

**注意事项**:
- QFN 底部需充分焊接
- 避免焊桥 (引脚间距 0.4mm)
- 检验需 X-Ray

### 7.3 测试要点

| 测试项 | 方法 | 判定标准 |
|--------|------|----------|
| 电源短路 | 万用表 | 无短路 |
| SPI 通信 | 读取 Device ID | 0xDECA03xx |
| RF 性能 | 网络分析仪 | S11 < -10dB |
| 功耗 | 电流表 | 符合规格 |

### 7.4 ESD 防护

- DW3220 RF 引脚敏感
- 操作需戴防静电手环
- 存储使用防静电袋
- PCB 设计增加 ESD 保护
